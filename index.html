RailsEventStore

RailsEventStore

A Ruby implementation of an EventStore. Default storage is events repository based on Active Record (provided by separate gem: rails_event_store_active_record).

Building the docs

Docs are available as Markdown files inside docs/ directory. They are processed by Gitbook to generate a nicely looking documentation book.
